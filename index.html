Submit all pending URLs (not yet sent to Bing/Yandex) to IndexNow in one call. This runs automatically on bulk submit, but you can trigger it manually here.

Step 1 — Environment setup
# 1. Install dependencies
pip install -r requirements.txt

# 2. Fill in .env file
# DATABASE_URL, SYNC_DATABASE_URL,
# REDIS_URL, INDEXNOW_KEY,
# INDEXNOW_HOST, GOOGLE_KEYS_DIR

# 3. Run migrations
alembic upgrade head

# 4. Start FastAPI (Terminal 1)
uvicorn app.main:app --reload

# 5. Start Celery worker (Terminal 2)
celery -A app.worker worker --loglevel=info --concurrency=4

# 6. Start Celery beat (Terminal 3)
celery -A app.worker beat --loglevel=info

# 7. Open dashboard at:
# http://localhost:8000
Step 2 — IndexNow setup (2 min)
# Set in .env:
INDEXNOW_KEY=any-uuid-you-choose
INDEXNOW_HOST=https://yourdomain.com

# Create a verification file at:
# https://yourdomain.com/{INDEXNOW_KEY}.txt
# File content: just the key on one line

# That's it. IndexNow is now active.
# Covers Bing + Yandex instantly.
# 10,000 URLs per API call, free.
Step 3 — Google service account setup
1.  Go to https://console.cloud.google.com
2.  Create a new project
3.  APIs & Services → Library → Search "Web Search Indexing API" → Enable
4.  IAM & Admin → Service Accounts → + Create Service Account
5.  Download the JSON key → place it in ./google_keys/
6.  Go to https://search.google.com/search-console
7.  Select your property → Settings → Users and Permissions → Add User
8.  Paste the service account email (client_email from the JSON file)
9.  Set permission to "Owner" → Save
10. Click "Scan for new accounts" in the Accounts tab above

Quota: 200 URLs / account / day
307 accounts × 200 = 61,400 URLs/day
Step 4 — XML sitemaps (automatic)
Sitemaps are generated automatically for each client_id.

URL format:
  http://localhost:8000/sitemaps/{client_id}
  http://localhost:8000/sitemaps/default

Submit to Google Search Console:
  Search Console → Sitemaps → Enter URL → Submit

Google will re-crawl on its own schedule.
No API quota consumed. Good passive backup.
